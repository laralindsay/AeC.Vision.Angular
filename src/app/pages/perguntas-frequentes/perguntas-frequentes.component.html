INSTALAÇÃO

Para efetuar a instalação do Vision 360 na maquina do operador e necessário seguir os seguintes passos:

1

Criar uma pasta no disco C com o nome "Monitoramento" na máquina do operador;

2

Copiar os arquivos da pasta: \\sophia\shared\Monitoramento\PRD\Instalador para a pasta criada no passo anterior;

3

Colocar o AeC.Automacao.MonitoramentoOperacional.exe para executar no agendador de tarefas, para que assim que o operador logar ele ative a aplicação;

OPERADOR NÃO APARECE NA HIERARQUIA?

Quando o operador não aparece na hierarquia ou aparece desconectado estando logado na máquina, siga os passos a seguir:

1

Verificar se o Vision 360 está instalado de forma correta na máquina. Para executar essa verificação, acionar a TI Local para que seja validado os pontos de Instalação.

2

Verificar se o operador esta logado com o seu usuário de rede. Para executar essa verificação, basta validar na máquina do operador se ele esta usando o usuário de rede dele.

3

Verificar se o segmento do operador está cadastrado no Vision 360. Para executar essa verificação, acessar a aba de segmentos em Configurações, caso não esteja cadatrado criar novo Segmento.

4

Caso o operador esteja logado com o usuário dele e mesmo assim não aparece na ferramenta, verificar com a Gestão de Acessos se o usuário de rede e o que está cadastrado no hominum.

5

Caso ele esteja com o usuário correto no Gestão de Acessos e ainda não aparece, verificar se operador esta com hierarquia correta no hominum.

6

Caso todos os passos acima forem validados e mesmo assim o operador não aparece, acionar o setor responsável pela ferramenta através de chamado no GestãoX.

CADASTRANDO UM SISTEMA

Para realizar o cadastro de um novo sistema, siga os passos a seguir:

1

Acessar a aba de Configurações/Sistemas;

2

Clicar em novo sistema;

3

Só deve ser colocado como padrão os sistemas que sejam comuns em todas as operações;

4

Para sistemas Desktop é necessário acrescentar uma descrição para a aplicação e o nome do processo executável da aplicação;

5

Para sistemas Web ou Diretorio é necessário acrescentar uma descrição para a aplicação e criar os titulos das aplicações

6

Para sistemas web, colocar apenas o prefixo de inicio, por exemplo "google.com.br", que o sistema considerará todos os valores que vierem na frente como devidos.

7

Para Diretórios, colocar apenas o prefixo de servidor e pasta, por exemplo "servidor\pasta_com_acesso_permitido", que o sistema considerará todos os arquivos dentro dessa pasta como devidos.

8

Para sistemas Desktop é necessário acrescentar uma descrição para a aplicação e o nome do processo executável da aplicação;

CADASTRANDO UM SEGMENTO

Para realizar o cadastro de um novo segmento, siga os passos a seguir:

1

Acessar a aba de Configurações/Segmentos;

2

Clicar em novo segmento;

3

O nome do segmento deve ser igual ao nome do segmento no hominum;

4

O tempo de inatividade de mouse e teclado é de no mínimo 40 segundos;

5

E necessário que seja selecionados os sistemas que não são padrões para todas as operações, caso o sistema não esteja na lista, cadastrar novo Sistema.

DESCRIÇÃO DE INDICADORES
Tempo Ausente

Somatório do período em que o operador esteve desconectado da máquina (caso trabalhe remotamente) ou com máquina bloqueada.

Tempo Inatividade

Somatório do período em que o operador esteve logado, com a máquina desbloqueada, mas sem atividade de mouse e teclado.

Tempo Em Sistemas Indevidos

Somatório do período em que o operador acessou um aplicativo ou site que não estava configurado dentre os que ele pode acessar.
